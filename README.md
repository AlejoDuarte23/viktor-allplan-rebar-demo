VIKTOR demo app for a simple pile cap visual rebar workflow in Allplan.

The app keeps the flow intentionally small:

- Parametrize a rectangular pile cap on four piles.
- Configure concrete cover, cap mat bars, pile vertical bars, and pile hoops.
- Review a clean 2D visual rebar sketch in a WebView.
- Review a calculated visual geometry schedule in a TableView.
- Send the same parameters to an Allplan PythonPart worker as visual geometry.
